### Unordered Lists

- Item 1
- Item 2

  - Nested item 2.2
- Item 3

1. First item
2. Second item
   1. Nested item 2.1

- [ ] Incomplete task
- [ ] Another task

## Code Blocks

```javascript
// JavaScript code block

  return `Hello, ${name}!`;
}

console.log(greet('World'));
```

```python
# Python code block
def greet(name):
    return f"Hello, {name}!"

print(greet("World"))
```
